Access real-time information with web search 
Give Claude web access to answer questions with current information and 
automatic source citations.

Track and verify information sources with citations 
Claude provides detailed references to sentences and passages from your source 
documents. Just provide documents and Claude handles source attribution 
automatically.

Give Claude domain-specific expertise with Skills 
Skills are reusable, filesystem-based resources that extend Claude’s 
functionality. Each Skill packages instructions, metadata, and optional 
resources that Claude uses automatically when relevant. 
Learn more 
<https://docs.claude.com/en/docs/agents-and-tools/agent-skills/overview>
