Welcome to the Claude Developer Platform 
You now have access to the Claude API. Here's the fastest path to making your 
first call: 






Get your API key 
Create your API key in the Claude Console 
<https://console.anthropic.com/settings/keys> and set it as an environment 
variable: 
export ANTHROPIC_API_KEY='your-api-key-here' 






Make your first call 
Choose your preferred client—cURL, Python, TypeScript, or Java. This 
quickstart guide <https://docs.claude.com/en/docs/get-started#c-url> provides 
example code for you to copy and run immediately.
